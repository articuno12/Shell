# Shell
A standalone shell 
<br>
# Features
- All basic commands
- Semicolon separated multiple commands
- Piping implemented
- Foreground and Background Processes
- List running Jobs
- Killing all background jobs
- exit and quit commands for exiting the shell
- Signal handling like Ctrl+D, Ctrl+Z and Ctrl+C
<br>
# Tested On
- gcc version 6.2.0
- Ubuntu 16.04
