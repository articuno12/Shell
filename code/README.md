# Instructions
<br>
#### To compile and run :
```make
./main
```
#### To run a command:
 ```
 <command> <argument if any> ;
 ```
#### For making a process background:
```
  <command> & ;
  ```
#### For piping :
 ```
 <command1> | <command2>
```

#### For redirection :
```
<command> </>/>> <file>
```
