# Instructions
Instructions for using this shell
#### To compile and run :
`make
./main
`
<br>
#### To run a command:
 `<command> <argument if any> ;`
 <br>
#### For making a process background:
` <command> & ;`
  <br>
#### For piping :
 `
 <command1> | <command2>
`
<br>
#### For redirection :
```
<command> </>/>> <file>
```
